# Sayak Rana – Portfolio

This repository contains a single-page portfolio website (`index.html`).

## Quick Deploy (GitHub Pages)
1. Create a new repository on GitHub. You can name it either:
   - `sayakrana.github.io` (user/organization site), or
   - any name you like (project site).
2. Upload `index.html` to the repository root and commit.
3. For project sites only: in **Settings → Pages**, set Source to **Deploy from a branch**, then choose `main` and `/ (root)` and save.
4. Your site will be live shortly at:
   - `https://sayakrana.github.io` (for user site), or
   - `https://<your-username>.github.io/<repo-name>` (for project site).
